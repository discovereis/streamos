📋 Deploy Guide — Get Online in 10 Minutes
① GitHub Pages (Frontend Dashboard)
1. Go to github.com → Create account → New Repository
2. Name it: streamos — check "Public"
3. Upload index.html (the dashboard file)
4. Settings → Pages → Source: main branch → Save
5. Your dashboard is live at: yourusername.github.io/streamos
② Railway.app (Backend — Free)
1. Go to railway.app → Sign up free with GitHub
2. New Project → Deploy from GitHub repo
3. Upload the backend/ folder as a new GitHub repo
4. Railway auto-detects Node.js and deploys it
5. Copy your Railway URL (e.g. streamos.up.railway.app)
6. Paste it above in "Backend Server URL" → Connect
③ API Keys Checklist
☐ YouTube API → console.cloud.google.com → Enable YouTube Data API v3
☐ Pexels → pexels.com/api → Free instant key
☐ ElevenLabs → elevenlabs.io → Free 10k chars/month
☐ OpenAI → platform.openai.com → $5 credit = 500+ videos
⚡ Total setup time: ~10 minutes. Once deployed, the backend runs 24/7 on Railway's free tier — auto-generating and scheduling videos even when your computer is completely off.
